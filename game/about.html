Our Goal

Our aim was to make a puzzle game with added mechanics from a couple of inspiring PC games to create a small mobile project. This game was made in support of RSPCA charity and is designed to run from our app in Thunkable X.

The Game: FunkoFox: Escape The Facility

FunkoFox: Escape The Facility is a 2D platforming game where you must escape a testing facility and your captor using only your portal gun to complete fun puzzles around a basic underground world. At the end of the game, you will reach the point of finishing off the last monster: the human who has been keeping you captured the whole time for animal testing. This theme was put in to relate to animal cruelty that happens in our world and needs to be eradicated!

To help the RSPCA anti-animal cruelty charity, you can donate to them from our app to raise awareness and bring other animals to the same rights as us humans.

Behind The Scenes

The game is mostly written in typescript using the Phaser 3 game engine. A Google cloud app engine server hosts the game code to provide to the frontend app made using Thunkable X.

Music

The main tracks in this game were written using a tracker for the Sega Megadrive and a digital audio workstation.
